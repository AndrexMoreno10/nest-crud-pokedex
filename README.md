# Ejecutar en Desarrollo

1. Clone Repository
2. Execute
```
yarn install
```
3. Have Nest CLI installed 
```
npm i -g @nestjs/
```

4. build/Run DB
```
docker-compose up -d
```


## Stack Used
* MongoDB
* NestJs
